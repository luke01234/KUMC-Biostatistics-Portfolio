This directory contains work on creating a set of heatmaps given our data on protein baits and preys

1) The Input csv contained in the "Input" directory, was run through the "heatmap.R" script in the "R" directory

2) The program creates 3 heatmaps, one on the raw data, one normalized, and another percentized

3) The Outputs were saved as HTML documents in the "Output" directory
